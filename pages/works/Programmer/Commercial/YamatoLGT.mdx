# Yamato Logistic Conference Rooms

I participated in the part of execution and closure of this project.

## Description

An audio & video system integrated using RTI for three rooms.

The video & audio of the rooms can be separated or combined in aid with the control system.

## Participation

### Execution

- Programming control systems using Integration Designer in RTi.

![YMT-ipad-1](/../public/pic/YAMATO-4.PNG)

- Creating graphical human interfaces for the user, such as VTproE in Crestron, and Integration Designer in RTi.

![YMT-ipad-2](/../public/pic/YAMATO-3.PNG)

### Closure (Continuous)

- Creating user manuals for functions and feature that the system provided.
- Providing user training on how to use the system.
- Performing after-sale on-site maintenance and remote troubleshooting services.
